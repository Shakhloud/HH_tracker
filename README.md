# HH Tracker Bot

Telegram бот для поиска и отслеживания вакансий на hh.ru с AI-анализом резюме и генерацией сопроводительных писем.

## Возможности

- **Поиск вакансий**: Поиск до 150 последних вакансий по вашему резюме
- **AI-анализ резюме**: Автоматический анализ навыков, опыта и предпочтений
- **Фильтрация по опыту**: Умная фильтрация вакансий по соответствию вашему опыту
- **Пагинация**: Просмотр всех найденных вакансий с удобной навигацией
- **Сортировка**: Сортировка по дате публикации или по соответствию
- **Рассылка уведомлений**: Автоматическая проверка новых вакансий каждые 20 минут
- **Генерация сопроводительных писем**: AI генерирует короткие персонализированные письма
- **Загрузка PDF**: Поддержка загрузки резюме в формате PDF

## Требования

- Docker и Docker Compose
- Telegram Bot Token
- OpenRouter API Key

## Установка

### 1. Клонирование репозитория

```bash
git clone <repository-url>
cd HH_tracker
```

### 2. Настройка переменных окружения

Создайте файл `.env` в корне проекта:

```env
# Telegram Bot
BOT_TOKEN=your_telegram_bot_token

# OpenRouter AI
OPENROUTER_API_KEY=your_openrouter_api_key
OPENROUTER_MODEL=nvidia/nemotron-3-nano-30b-a3b:free

# Database
DATABASE_URL=postgresql+asyncpg://user:password@db:5432/hh_tracker

# Redis
REDIS_URL=redis://redis:6379/0

# Proxy (опционально, для регионов с блокировкой Telegram)
PROXY_URL=tg://proxy?server=your_proxy_server&port=8443&secret=your_secret

# HH API
HH_API_URL=https://api.hh.ru
HH_USER_AGENT=HH-Tracker-Bot/1.0

# Scheduler
CHECK_INTERVAL_MINUTES=20
MAX_VACANCIES=150
```

### 3. Запуск

```bash
docker-compose up -d
```

Бот будет доступен и начнёт обрабатывать сообщения.

## Использование

### Первый запуск

1. Отправьте боту команду `/start`
2. Выберите способ добавления резюме:
   - **Загрузить PDF** - отправьте PDF файл с резюме
3. Дождитесь анализа резюме AI
4. После анализа откроется главное меню

### Главное меню

- **🔍 Найти вакансии** - Поиск вакансий по вашему резюме
- **🔔 Подписаться на рассылку** / **🔕 Отписаться** - Управление уведомлениями
- **📤 Загрузить PDF резюме** - Обновить резюме новым PDF файлом
- **ℹ️ Помощь** - Показать справку

### Поиск вакансий

1. Нажмите **🔍 Найти вакансии**
2. Бот найдёт до 150 вакансий и отфильтрует по соответствию вашему опыту
3. Покажет 10 самых свежих вакансий с процентом соответствия
4. Используйте кнопки навигации:
   - **◀️ Назад** / **Вперед ▶️** - Листать страницы
   - **🎯 Сортировать по соответствию** - Показать топ-10 по релевантности
   - **📅 Сортировать по дате** - Вернуться к сортировке по дате

### Просмотр вакансии

1. Нажмите на название вакансии в списке
2. Откроется подробная информация:
   - Название и компания
   - Зарплата
   - Требования и обязанности
   - Процент соответствия вашему резюме
   - Ссылка на вакансию
3. Доступные действия:
   - **📝 Сгенерировать сопроводительное письмо** - AI создаст короткое письмо (2-3 предложения)
   - **🔙 Назад к списку** - Вернуться к списку вакансий

### Генерация сопроводительного письма

1. При просмотре вакансии нажмите **📝 Сгенерировать сопроводительное письмо**
2. AI проанализирует ваше резюме и требования вакансии
3. Получите короткое персонализированное письмо (2-3 предложения)
4. Письмо будет в блоке `<pre>` для удобного копирования
5. Также будет ссылка на вакансию
6. Можно сгенерировать заново, если результат не понравился

### Рассылка уведомлений

1. Нажмите **🔔 Подписаться на рассылку** в главном меню
2. Бот будет проверять новые вакансии каждые 20 минут
3. При появлении подходящих вакансий придёт уведомление с количеством
4. Нажмите **📋 Показать все** в уведомлении, чтобы увидеть список
5. Для отмены нажмите **🔕 Отписаться от рассылки**

### Обновление резюме

1. Нажмите **📤 Загрузить PDF резюме** в главном меню
2. Отправьте новый PDF файл
3. Дождитесь анализа
4. Данные резюме обновятся автоматически

## Технические детали

### Архитектура

- **Python 3.11** - Язык программирования
- **aiogram 3.x** - Фреймворк для Telegram Bot API
- **SQLAlchemy 2.0** + **asyncpg** - Асинхронная работа с PostgreSQL
- **aiohttp** - Асинхронные HTTP запросы
- **OpenRouter API** - AI для анализа резюме и генерации писем
- **Docker** - Контейнеризация

### Структура проекта

```
HH_tracker/
├── bot.py                 # Основной файл бота, обработчики
├── config.py             # Конфигурация и настройки
├── database.py           # Модели базы данных
├── hh_api.py            # Интеграция с API hh.ru
├── resume_analyzer.py   # AI-анализ резюме
├── pdf_parser.py        # Парсинг PDF файлов
├── scheduler.py         # Планировщик проверок
├── requirements.txt     # Зависимости Python
├── Dockerfile          # Конфигурация Docker
├── docker-compose.yml  # Docker Compose конфигурация
└── .env               # Переменные окружения
```

### База данных

**Таблица `users`**:
- `telegram_id` - ID пользователя в Telegram
- `resume_text` - Текст резюме
- `skills` - Извлечённые навыки (JSON)
- `experience_years` - Опыт работы в годах
- `desired_position` - Желаемая позиция
- `location` - Локация
- `salary_expectation` - Ожидаемая зарплата
- `is_subscribed` - Статус подписки на рассылку

**Таблица `vacancies`**:
- `hh_id` - ID вакансии на hh.ru
- `name` - Название вакансии
- `employer_name` - Название компании
- `url` - Ссылка на вакансию
- `salary_from`, `salary_to`, `salary_currency` - Зарплата
- `location` - Локация
- `experience` - Требуемый опыт
- `published_at` - Дата публикации

### AI Модели

По умолчанию используется `nvidia/nemotron-3-nano-30b-a3b:free` - бесплатная модель от NVIDIA на OpenRouter.

Можно изменить в `.env`:
- `google/gemma-2-9b-it:free` - Google Gemma
- `meta-llama/llama-3.1-8b-instruct:free` - Meta Llama

### Фильтрация вакансий

Бот фильтрует вакансии по опыту работы:
- Если у вас 1 год опыта - не покажет вакансии требующие 3+ года
- Соответствие считается по навыкам, позиции и опыту
- Минимальный порог соответствия - вакансии с низким соответствием отфильтровываются

## Устранение неполадок

### Ошибка 401 при анализе резюме

- Проверьте `OPENROUTER_API_KEY` в `.env`
- Получите новый ключ на https://openrouter.ai/keys

### Ошибка 404 при анализе

- Модель AI не найдена
- Проверьте `OPENROUTER_MODEL` в `.env`
- Используйте одну из бесплатных моделей

### Бот не отвечает

```bash
# Проверить статус контейнеров
docker-compose ps

# Посмотреть логи
docker logs hh_tracker-bot-1

# Перезапустить
docker-compose restart bot
```

### Ошибка подключения к Telegram

Если вы в регионе с блокировкой Telegram, настройте прокси:
```env
PROXY_URL=tg://proxy?server=your_server&port=8443&secret=your_secret
```
